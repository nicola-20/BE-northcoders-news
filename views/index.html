Getting started

Take a look at the following endpoints to find out what you can do inside the NC News API

/api

/api

Brings you to this very page!

NOTE: * Endpoint can be sorted via a query; e.g. /comments?sort=votes&by=asc, /articles?sort=created_at&by=desc

/topics

/topics

GET: Get all the topics*

/topics/:topic_slug/articles

GET: Get all the articles* for a certain topic

POST: Add a new article to a topic. This requires a JSON with title and body

/users

/users

GET: Get all the users*

/users/:username

GET: Get a user by their username

/articles

/articles

GET: Get all the articles*

/articles/:article_id

GET: Get an individual article

PATCH: Increment or decrement the votes of an article by one using a query. e.g. ?vote=up, ?vote=down

/articles/:article_id/comments

GET: Get all the comments* for an individual article

POST: Add a new comment to an article. This requires a JSON object with body and created_by

/comments

/comments

GET: Get all the comments*

/comments/:comment_id

GET: Get a comment by its ID

PATCH: Increment of decrement the votes of a comment by one. Requires a query of vote = up or down. e.g. ?vote=up

DELETE: Deletes a comment
